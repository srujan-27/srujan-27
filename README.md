

### About

I'm an AI/ML Engineer with 4+ years building and shipping production machine learning, deep learning, and generative AI systems across banking, telecom, and fintech. Most of my recent work lives in the LLM stack: RAG pipelines, agentic workflows, fine-tuning with PEFT/LoRA, and vector search with FAISS and Pinecone, plus the MLOps around it on AWS, Azure, and GCP using Docker, Kubernetes, and MLflow.

I also build full-stack apps with the MERN stack, which is a big part of why the ML systems I ship tend to actually run in production rather than dying in a notebook.

Currently finishing my M.S. in Computer Science at Quinnipiac University (Dec 2025).
